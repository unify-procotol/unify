# Unify

A Hono-based SDK with built-in Web3 support for unified blockchain data access

## Overview

Unify is a powerful and flexible SDK built on top of the high-performance [Hono](https://hono.dev/) web framework. It provides unified access to multiple blockchain networks through a consistent API, making it easy to build Web3 applications and query blockchain data.
